Розробка на стороні сервера

Серверна розробка передбачає написання бекенд логіки для веб додатків на мовах програмування таких як JavaScript, Python, PHP, C# та інших. Розглянемо основи.

Огляд архітектури веб-додатків

Огляд клієнт-серверної архітектури, ролі веб-сервера, запит/відповідь, REST API та основ проектування бекенду.

Node.js

Платформа на основі JavaScript для написання серверної логіки. Асинхронне виконання, модулі, роутинг, middleware.

Використання баз даних у веб-додатках

Взаємодія веб-додатків з реляційними та NoSQL базами даних. SQL, MongoDB та інші.

Така структура дозволяє дати базове уявлення про серверну розробку веб-додатків та її ключові аспекти.
