Розгортання та хостинг веб-додатків

Після розробки, веб-додатки потрібно розгорнути та розмістити на публічних серверах чи хмарних платформах.

Використання хмарних платформ

Огляд хмарних провайдерів, таких як AWS, Google Cloud, Microsoft Azure. Створення віртуальних машин, контейнерів та баз даних.

Контейнеризація та Docker

Пакування додатків у контейнери, локальний запуск та оркестрація на виділеному сервері чи в хмарі. Docker і альтернативи.

CI/CD пайплайни

Безперервна інтеграція та безперервне розгортання за допомогою автоматизованих пайплайнів на GitHub Actions, GitLab CI та інших.
