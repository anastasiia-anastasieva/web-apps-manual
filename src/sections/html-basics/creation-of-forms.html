Створення форм

HTML форми дозволяють збирати дані від користувачів для подальшої обробки на сервері. Форма створюється за допомогою тегу <form>:

<form action="/handling-form.php" method="post">
</form>

Атрибут action вказує куди надсилатимуться дані форми, method - метод надсилання (get або post).

Елементи форми

Всередині форми додаються елементи для вводу даних. Наприклад:

  • <input> - різні типи полів вводу

  • <textarea> - багаторядкове текстове поле

  • <select> - випадаючий список

Приклад форми з полями для імені, email, повідомлення та кнопкою відправки:


<form action="/submit-form.php" method="post">
    <label for="name">Ім'я:</label>
    <input type="text" id="name" name="name">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <label for="message">Повідомлення:</label>
    <textarea id="message" name="message"></textarea>
    <input type="submit" value="Відправити">
</form>

Таким чином можна створювати різноманітні форми для збору корисної інформації від відвідувачів сайту.

Практичне завдання

Щоб краще зрозуміти роботу з HTML формами, спробуйте виконати наступне:

  • Створіть HTML сторінку та додайте форму з атрибутами action і method

  • Додайте поле для введення імені та email:

    • <label>Ім'я</label>
<input type="text" name="username">

<label>Email</label>
<input type="email" name="email">

  • Створіть текстову область для введення повідомлення:

    • <label>Повідомлення</label>
<textarea name="message"></textarea>

  • Додайте checkbox для згоди на обробку даних:

    • <input type="checkbox" name="consent">
<label>Я згоден з обробкою моїх даних</label>

  • Додайте кнопку відправки форми:

    • <input type="submit" value="Відправити">

  • Перевірте роботу форми, заповнивши її та натиснувши на кнопку.

  • Спробуйте додати інші типи полів, такі як radio, select, дата.
