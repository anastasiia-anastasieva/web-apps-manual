Елементи, теги та атрибути

HTML документ складається з елементів. Елементи позначаються тегами та мають атрибути.

Теги

Теги - це ключові слова, обгорнуті в кутові дужки: <тег> та </тег>. Вони визначають початок та кінець елементу.

Наприклад, тег <p> позначає початок параграфу, а </p> - його кінець. Атрибути вказують додаткову інформацію про елемент, наприклад, клас або ідентифікатор.

Атрибути

Атрибути надають додаткову інформацію про елемент та додаються до початкового тегу після пробілу.

Наприклад, <a href="https://example.com"> містить атрибут href, який вказує посилання.

Основні атрибути:

  • id - унікальний ідентифікатор елемента

  • class - визначає клас елемента

  • src - шлях до зовнішнього ресурсу (зображення, скрипт)

  • href - URL-адреса посилання

  • style - вбудовані CSS стилі

Знання елементів, тегів та атрибутів дозволяє гнучко структурувати та стилізувати HTML документ.

Практичне завдання

Щоб краще зрозуміти роботу з елементами, тегами та атрибутами, спробуйте наступне:

  • Створіть HTML сторінку та додайте заголовок першого рівня: <h1>Мій заголовок</h1>

  • Додайте абзац тексту і присвойте йому унікальний id:

    • <p id="intro">Тут іде абзац вступного тексту...</p>

  • Додайте ще один абзац і присвойте йому клас highlight:

    • <p class="highlight">Цей абзац потрібно виділити</p>

  • Додайте ще один абзац і присвойте йому клас highlight:

    • <p class="highlight">Цей абзац потрібно виділити</p>

  • Вставте зображення, вказавши URL в атрибуті src:

    • <img src="image.jpg" alt="Зображення">

  • Додайте посилання з відповідним значенням в атрибуті href:

    • <a href="https://example.com">Посилання на приклад</a>

  • Перегляньте результат у браузері та переконайтеся, що все працює правильно.

  • Поекспериментуйте з додаванням інших атрибутів та стилів.

Це допоможе закріпити розуміння основних конструкцій HTML.
