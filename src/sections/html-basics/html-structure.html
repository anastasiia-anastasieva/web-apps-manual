Структура HTML-документа

Розглянемо базову структуру HTML документа та його основні елементи.

DOCTYPE

Оголошення <!DOCTYPE> - є визначення типу документа, яке допомагає браузерам правильно відображати вебсторінки. Воно повинно з'являтися тільки один раз у верхній частині вебсторінки (перед будь-якими тегами HTML). Оголошення <!DOCTYPE> не чутливе до регістру. Можна писати як великими літерами, так і маленькими. Оголошення <!DOCTYPE> для HTML5 має такий вигляд:

<!DOCTYPE html>

Нижче наведена візуалізація структури HTML-сторінки:

html

Елемент <html> є кореневим елементом HTML сторінки. Всі інші елементи мають бути всередині тегів <html> та </html>.

head

Елемент <head> містить метадані про документ - інформацію про сам документ, а не його вміст. Сюди входять такі елементи:

  • <title> - заголовок сторінки, який відображається в заголовку вікна браузера або на вкладці.

  • <meta> - метатеги, які містять інформацію про сторінку, таку як кодування, опис, ключові слова, автор, та інше.

  • <link> - посилання на зовнішні ресурси, такі як CSS-файли, шрифти, та інше.

  • <style> - внутрішній CSS-код, який використовується для задання стилів для сторінки.

body

Елемент <body> містить весь видимий вміст сторінки - текст, зображення, посилання, список, таблиці тощо.

Приклад

Ось приклад простої структури HTML документа:

<!DOCTYPE html>
<html>
    <head>
<title>Заголовок сторінки</title>
    </head>
    <body>
        <h1>Це заголовок сторінки</h1>
        <p>Це параграф з текстом.</p>
    </body>
</html>

Висновок

Розглянули основні елементи структури HTML документа. Використовуючи правильну семантичну розмітку, ви можете створювати доступні та оптимізовані веб-сторінки.

Практичне завдання

Щоб краще зрозуміти структуру HTML документа, спробуйте виконати наступне:

  • Створіть новий HTML файл з ім'ям structure.html

  • Додайте оголошення DOCTYPE для HTML5

  • Додайте основні HTML теги <html>, <head>, та <body>

  • Всередині <head> додайте елемент <title>, який міститиме назву вашої сторінки

  • Всередині <body> додайте заголовок <h1>, абзац <p>, зображення <img> та посилання <a>

  • Перегляньте сторінку в браузері та переконайтеся, що вміст відображається правильно

  • Валідуйте код за допомогою checkValidity та виправте можливі помилки
