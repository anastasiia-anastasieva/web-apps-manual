HTML

HTML (HyperText Markup Language) є основою будь-якого веб-сайту. Це не програмна мова, а мова розмітки, яка визначає структуру веб-сторінок.

Використовуючи HTML, можна створювати і організовувати різні частини веб-сторінки, такі як заголовки, параграфи, списки, посилання, зображення, та багато іншого.

HTML-документи складаються з набору елементів, які вказують браузеру, як відображати вміст. Кожен елемент може мати атрибути, що надають додаткову інформацію про елемент або змінюють його поведінку чи зовнішній вигляд.

Підрозділи:

Структура HTML - документа

Дізнаємося, як правильно організувати вміст веб-сторінки та використовувати метадані для вказівки заголовку сторінки, опису, ключових слів, та іншого.

Елементи, теги та атрибути

Поглибимо знання про елементи HTML, дізнаємося про різницю між блоковими та інлайновими елементами, розглянемо основні теги та їх призначення.

Створення форм

Навчимося створювати форми за допомогою HTML, використовуючи різні типи полів введення (input), такі як текст, пароль, радіокнопки, чекбокси, випадаючі списки та кнопки.

Кожен з цих підрозділів містить теоретичні матеріали, приклади коду та тестові завдання, щоб ви могли не тільки прочитати про HTML, а й відразу застосувати отримані знання на практиці.
