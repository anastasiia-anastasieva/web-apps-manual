Управління потоком

Умовні оператори

Іноді нам потрібно виконувати різні дії на основі різних умов. Для цього ми можемо використовувати інструкцію if та умовний оператор ?, що також називається оператором “знак питання”.

Інструкція “if”

Інструкція if(...) обчислює умову в дужках і, якщо результат умови true, виконує блок коду.
Наприклад:

let year = prompt('В якому році була опублікована специфікація ECMAScript-2015?', '');

if (year == 2015) alert( 'Ви маєте рацію!' );

У наведеному вище прикладі умовою є проста перевірка рівності (year == 2015), але умова може бути набагато складнішою.

Якщо ми хочемо виконати більше однієї інструкції, ми повинні загорнути блок коду у фігурні дужки:

if (year == 2015) {
  alert( "Це правильно!" );
  alert( "Ви такий розумний!" );
}

Рекомендуємо огортати блок коду фігурними дужками {} кожного разу, коли ви використовуєте інструкцію if, навіть якщо виконуватиметься лише одна операція. Це покращує читабельність.

Булеве перетворення

Інструкція if (…) обчислює вираз у дужках і перетворює результат у логічний тип.

Правила перетворення

  • Число 0, порожній рядок "", null, undefined та NaN перетворюються на false. Через це їх називають “хибними” (“falsy”) значеннями.
  • Інші значення перетворюються на true, тому їх називають “правдивими”.

Отже, код ніколи не виконається за такої умови:

if (0) { // 0 є хибним
  ...
}

…і завжди виконається за такої:

if (1) { // 1 є правдивим
  ...
}

Блок “else”

Вираз if може містити необов’язковий блок else (“інакше”). Він виконується, коли умова є хибною.
Наприклад:

            let year = prompt('В якому році була опублікована специфікація ECMAScript-2015?', '');

if (year == 2015) {
  alert( 'Ви здогадалися правильно!' );
} else {
  alert( 'Як ви можете так помилятися?' ); // будь-яке значення, окрім 2015
}

Цикли

while виконує блок коду, поки умова істинна:

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

for виконує блок задану кількість разів:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Цикли можна переривати операторами break та continue.

Умовні оператори та цикли дозволяють керувати потоком виконання програми.

Практичне завдання

  • Напишіть програму, яка за допомогою циклу виводить числа від 1 до 10.

  • Напишіть програму, яка за допомогою циклу виводить лише парні числа від 2 до 20.

  • Напишіть програму, яка виводить числа від 10 до 1, використовуючи цикл та оператор декременту.

  • Напишіть програму, яка за допомогою циклу знаходить суму всіх чисел від 1 до 100.

  • Напишіть програму, яка за допомогою циклу знаходить найбільше число в масиві.

  • Напишіть програму, яка за допомогою умовного оператора перевіряє чи число додатнє, від’ємне або дорівнює нулю.

  • Напишіть програму, яка за допомогою умовного оператора перевіряє чи рік високосний.

  • Напишіть програму, яка виводить привітання ("Добрий день" чи "Добрий вечір") залежно від значення змінної hour.

Такі завдання допоможуть закріпити навички управління потоком програми в JavaScript за допомогою умовних операторів та циклів.
