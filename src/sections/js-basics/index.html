JavaScript

JavaScript є мовою програмування для створення інтерактивних веб-сторінок. Розглянемо основи JavaScript.

Синтаксис і типи даних

Змінні та присвоєння значень, основні типи даних: числа, рядки, логічні значення. Масиви та об'єкти. Оператори порівняння та логічні оператори

Управління потоком

Умовні оператори if/else. Цикли while та for. Оператори break та continue

Обробка подій

Визначення функцій та виклик. Підключення функцій до подій. this та область видимості. Анонімні функції та колбеки

Детальніше розглянемо ці та інші теми JavaScript з прикладами коду.
