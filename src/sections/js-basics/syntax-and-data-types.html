Синтаксис і типи даних

JS синтаксис

Синтаксис JavaScript - це набір правил, згідно яким створюються програми JavaScript:

// Як створювати змінні:
var x;
let y;

// Як використовувати змінні:
x = 5;
y = 6;
let z = x + y;

Синтаксис JavaScript визначає два типи значень: фіксовані значення та значення змінних. Фіксовані значення називаються літералами. Значення змінних називаються змінними.

JavaScript Літерали

Найважливіші правила для запису фіксованих значень:

Числа пишуться з десятковими знаками або без них:

Також можна змінювати розміри і розташування блоків в залежності від ширини екрану:

10.50

1001

Рядки - це текст, написаний у подвійних або одинарних лапках:

"John Doe"

'John Doe'

JavaScript Змінні

У мові програмування змінні використовуються для зберігання значень даних. JavaScript використовує ключове слово var для оголошення змінних (від анг. variable - змінна). Знак рівності використовується для присвоєння значень змінним.
У цьому прикладі x визначається як змінна. Потім х надається значення 6:

var x;

x = 6;

Основні типи даних:

  • Числа - цілі або дробові значення.
  • Рядки - текст в лапках: 'text' або "text".
  • Логічні значення - true або false.

Масиви та об'єкти

Масив - список даних:

var cars = ["Saab", "Volvo", "BMW"];

Об'єкт - набір властивостей та значень:

var person = {
 firstName:"John",
 lastName:"Doe",
 age:50,
 eyeColor:"blue"
};

Оператори порівняння та логічні оператори

Оператори порівняння: >, <, ===, !== тощо.
Логічні оператори: &&, ||, !.
Використовуються в умовних конструкціях.

Практичне завдання

  • Оголосіть змінні різних типів - число, рядок, логічне значення. Виведіть їх значення в консоль.

  • Створіть масив з кількома елементами різних типів. Виведіть перший та останній елементи.

  • Створіть об'єкт "користувач" з властивостями ім'я, вік, email. Виведіть значення властивостей.

  • Напишіть функцію, яка приймає два числа і повертає більше з них, використовуючи оператор порівняння.

  • Напишіть функцію, яка приймає число і повертає рядок "парне" або "непарне" в залежності від числа, використовуючи логічний оператор.

  • Створіть об'єкт "книга" з властивостями назва, автор, рік. Створіть масив з кількох таких об'єктів.

  • Напишіть функцію, яка приймає масив чисел і повертає новий масив, де кожен елемент збільшений на 1.

  • Напишіть функцію, яка приймає масив чисел і повертає новий масив, де кожен елемент збільшений на 1.
