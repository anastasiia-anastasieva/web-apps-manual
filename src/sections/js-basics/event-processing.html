Обробка подій

В JavaScript обробка подій відбувається за допомогою функцій-слухачів, що підписуються на певні події.

Визначення функцій та виклик

Функції визначаються ключовим словом function:

function greet() {
  alert('Hello!');
}

І викликаються по імені:

greet();

Підключення функцій до подій

Наприклад, підписка на подію "click":

button.addEventListener('click', function() {
  // код при кліку
});

Тут анонімна функція буде викликатися при настанні події "click" на елементі button.

this та область видимості

В обробнику подій this посилається на елемент, на якому сталася подія. Змінні всередині функції локальні. Зовнішні змінні доступні через замикання. Обробка подій дозволяє реагувати на дії користувача та оживляти веб-сторінки.

Практичне завдання

  • Створіть HTML-сторінку з кнопкою та блоком тексту.

  • Додайте обробник на подію "click" для кнопки, який буде виводити текст у блок.

  • Створіть функцію, яка змінює колір фону сторінки на випадковий при натисканні кнопки.

  • Додайте обробку події "input" для текстового поля, яка буде виводити його поточне значення.

  • Створіть таймер, який кожну секунду виводить поточний час на сторінку.

  • Додайте обробку події "resize" для вікна браузера, яка виводить його поточну ширину.

  • Створіть галерею зображень, де при кліку на мініатюру відбувається заміна великого зображення.

  • Додайте обробку події submit для форми, яка буде виводити значення полів.
