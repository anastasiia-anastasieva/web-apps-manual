Electron

Electron - це фреймворк для створення desktop додатків з використанням HTML, CSS та JavaScript.

Основи Electron

Electron - це фреймворк на основі Node.js та Chromium для створення настільних додатків за допомогою веб-технологій.
Основні особливості Electron:

  • Кросплатформовість

  • Можливості Node.js в додатку

  • Інтеграція з OS

  • Нативний користувацький інтерфейс

Структура додатків складається з:

  • Main процесу - відповідає за створення вікон, має доступ до OS API. Написаний на Node.js.

  • Render процесів - створюють сторінки у вікнах. Використовують HTML, CSS та JS.

function Hello() {
  return <h1>Hello!</h1>
}

Таким чином ви можете створити кілька вікон з окремими Render процесами для кожного. Між ними відбувається спілкування через IPC (InterProcess Communication).

Інтерфейс будується стандартними веб-технологіями (HTML/CSS/JS) в Renderer процесі, в той час як Main процес відповідає за інтеграцію з операційною системою.

Перші кроки

Щоб почати створювати Electron додаток вам потрібно встановити Node.js та пройти ініціалізацію проекту через npm init.

Далі ви зможете створювати основне файл додатку main.js та html файл для інтерфейсу index.html.
Зв'язок між ними відбувається через модуль electron, який дозволяє створювати вікна та керувати життєвим циклом.

Розробка інтерфейсу

Інтерфейс в Електроні робиться за допомогою звичайних веб-технологій:

const App = () => {
  return (
    <Window>
      <MainMenu />

      <Content>
        // UI code
      </Content>
    </Window>
  )
}

Тобто React чи інша бібліотека підійде.

Electron дозволяє пакувати веб-додатки як повноцінні нативні програми для різних платформ.

Практичне завдання

  • Ініціалізуйте новий проект за допомогою npm init

  • Встановіть пакети electron та electron-builder через npm

  • Створіть файл main.js з наступним кодом:

    • const {app, BrowserWindow} = require('electron')

let win;

app.on('ready', () => {
  win = new BrowserWindow();
  win.loadFile('index.html');
})

  • Створіть index.html файл з текстом "Hello World"

  • Запустіть додаток командою npm start

  • Спробуйте показати реальне вікно з кнопками за допомогою HTML і CSS

  • Додайте функціонал взаємодії з вікнами та контентом

  • Відпрацюйте build та дистрибуцію готового пакету додатку

  • Опублікуйте додаток в магазині GitHub або іншому
