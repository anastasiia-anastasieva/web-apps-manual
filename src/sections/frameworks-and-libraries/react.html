React

React - це бібліотека для створення інтерфейсів з використанням компонентного підходу.

JSX

JSX дозволяє писати розмітку HTML всередині коду JavaScript файлів. Він транспілюється в виклики функцій React.

function Hello() {
  return <h1>Hello!</h1>
}

Компоненти

Основна ідея React - це компоненти - окремі маленькі частини інтерфейсу:

function Button() {

  return (
    <button>some button</button>
  );
}

Компоненти приймають пропси та повертають реакт елементи.

Стан та життєвий цикл

Використовуючи стан (state) та хуки життєвого циклу, можна зробити компоненти динамічними та інтерактивними.
Overall, React дозволяет створювати комплексні інтерфейси з ізольованими модульними компонентами.

Практичне завдання

  • Створіть простий реакт додаток за допомогою create-react-app

  • Створіть функціональний компонент Greeting, який рендерить елемент привітання <h1>

  • Створіть компонент Message, який приймає текст через props та виводить його

  • Викличте компоненти Greeting та Message у компоненті App

  • Створіть стан в компоненті App, що зберігатиме повідомлення

  • Передайте стан як пропс в компонент Message

  • Додайте обробку подій та оновлення стану по натисканню кнопок

  • Відобразіть динамічне оновлення компонента Message при зміні стану

  • Додайте CSS стилі для компонентів

  • Збільдіть проект та перевірте його роботу
