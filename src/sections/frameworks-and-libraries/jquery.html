jQuery

Query - популярна JavaScript бібліотека для спрощення роботи з DOM, подіями, AJAX запитами та іншим.

Основи jQuery

Щоб почати використовувати jQuery потрібно підключити бібліотеку:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

Далі можна вибирати елементи за допомогою функції $():

$('button'); // вибір за селектором
$('#submit'); // вибір за ID

Робота з елементами

jQuery надає широкі можливості:

$('p').hide(); // приховати елементи
$('h1').css('color', 'red'); // задати CSS стиль
$('form').submit(function() {}); // подія на формі

AJAX запити

Для відправки AJAX запитів використовується функція $.ajax() або спеціалізовані методи:

$.get('/api/users', data => {
  // callback функція
});

jQuery значно спрощує JavaScript розробку за рахунок зручного інтерфейсу для роботи з DOM та подіями.

Практичне завдання

  • Підключіть бібліотеку jQuery до HTML документа

  • Використовуючи jQuery селектори, отримайте елемент з ID "content" та елементи класу "paragraph"

  • За допомогою jQuery приховайте або покажіть якийсь блок на сторінці

  • Змініть колір фону для деяких елементів за допомогою методу css()

  • Додайте клас для виділення парних елементів списку, використовуючи метод addClass()

  • Реалізуйте слайдер з зображеннями, що змінюються при кліках на кнопки "Next"/"Prev"

  • Зробіть ajax запит і виведіть JSON дані із сервера в якийсь блок на сторінці

  • Додайте анімацію за допомогою методів animate() або fadeOut()

  • Обробіть подію submit на формі та виведіть дані форми
