Адаптивний дизайн із застосуванням медіа-запитів

Адаптивний дизайн дозволяє створювати веб-сторінки, які добре виглядають на різних пристроях з різними розмірами екрану. Для цього використовуються медіа-запити в CSS.

Медіа-запити для умовного CSS

Медіа-запити дозволяють задавати стилі, які будуть застосовуватись тільки при певних умовах, наприклад, при певній ширині екрану:

@media screen and (max-width: 768px) {
  /* Стилі для екранів до 768px */
}

Макети для різних розмірів екранів

За допомогою медіа-запитів можна створювати різні макети для різних розмірів екранів. Наприклад, для мобільних пристроїв можна показувати меню, яке розкривається, а для десктопів - горизонтальне меню.

Також можна змінювати розміри і розташування блоків в залежності від ширини екрану:

@media screen and (min-width: 1200px) {
  .container {
    width: 1170px;
  }
}

@media screen and (max-width: 1199px) {
  .container {
    width: 970px;
  }
}

@media screen and (max-width: 991px) {
  .container {
    width: 750px;
  }
}

Резинова верстка за допомогою flex та grid

Flex та grid в CSS дозволяють створювати гнучкі макети, які адаптуються до розмірів екрану.
Flex дозволяє розміщувати елементи в рядок або стовпець і розподіляти між ними вільний простір:

.row {
  display: flex;
  flex-wrap: wrap;
}

.col {
  flex: 1;
  padding: 10px;
}

Grid дозволяє створювати двовимірні макети з рядками і стовпцями:

.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  grid-gap: 20px;
}

Разом медіа-запити, flex та grid дозволяють створювати адаптивні веб-сторінки, які добре виглядають на будь-яких пристроях.

Практичне завдання

  • Створіть просту HTML сторінку з шапкою, основним контентом та футером.

  • Додайте базові стилі з фіксованими розмірами блоків.

  • Створіть медіа-запит для мобільних пристроїв і змініть розміри блоків під них.

  • Додайте ще кілька точок перелому для планшетів та десктопів.

  • Використайте flexbox для створення адаптивного рядка з карточками, які вміщуються в один рядок.

  • Застосуйте CSS grid для адаптивної сітки зображень, яка змінює кількість стовпців.

  • Перевірте, як виглядає сторінка на різних пристроях і розмірах вікна браузера.

  • Додайте необхідні правки для поліпшення адаптивності.

Таке завдання допоможе закріпити навички створення адаптивного дизайну з CSS.
