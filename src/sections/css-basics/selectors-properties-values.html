Селектори, властивості та значення

Анатомія набору правил CSS

Вся структура називається набором правил (але найчастіше для стислості "правило"). Зазначимо також імена окремих частин:

  • Селектор (Selector) - вказує на які елементи буде застосовуватися стиль.

  • Оголошення (Declaration) - вказує, які властивості елемента ви хочете стилізувати.

  • Властивість (Property) - параметр, який ми хочемо змінити.

  • Значення (Value) - значення, яке ми хочемо присвоїти властивості.

Селектори використовуються в CSS для того, щоб вказати до яких елементів застосовуватимуться стилі. Існують різні типи селекторів:

  • За типом тегу - h1 { }

  • За класом - .className { }

  • За ID - #uniqueId { }

  • Комбіновані - div.post p { }

Властивості в CSS відповідають за вибір параметру для стилізації, наприклад color, font-size, padding тощо.

Значення присвоюються властивостям і визначають, як саме буде застосований цей стиль:

h1 {
    color: red;
    font-size: 24px;
}
}

Тут color та font-size - властивості, а blue та 20px - їх значення.

Каскадність та специфічність в CSS дозволяє перевизначати стилі і надавати пріоритет тим чи іншим правилам.

Отже, вміле використання різних типів селекторів, властивостей та значень дає широкі можливості для стилізації в HTML.

Практичне завдання

  • Створіть HTML документ з різними тегами - заголовки h1-h6, абзаци p, список ul/li тощо.

  • Підключіть зовнішній CSS файл.

  • Використовуючи селектор тегу задайте колір та розмір заголовків h1-h3.

  • За допомогою класу задайте стилі для виділення деяких абзаців.

  • Через ID задайте унікальні стилі для конкретного елементу.

  • Використайте комбінований селектор для стилізації li всередині ul.

  • Додайте різні властивості та значення, такі як колір, розмір тексту, відступи тощо.

  • Перегляньте результат та переконайтеся, що стилі застосувались правильно.

  • Поекспериментуйте з каскадністю та специфічністю селекторів.

Таке завдання допоможе краще зрозуміти принципи роботи з CSS селекторами, властивостями та значеннями.
