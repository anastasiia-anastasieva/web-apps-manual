CSS (Каскадні таблиці стилів)

CSS (Cascading Style Sheets) — це код, який ви використовуєте для стилізації веб-сторінки. Основи CSS допоможуть вам зрозуміти, що потрібно для початку роботи.

CSS має різні рівні та профілі. Наступний рівень CSS створюється на основі попередніх, додаючи нову функціональність або розширюючи функції, що вже існують. Рівні позначаються як CSS1, CSS2 та CSS3. Профілі - сукупність правил CSS одного або більше рівнів, створені для окремих типів пристроїв або інтерфейсів. Наприклад, існують профілі CSS для принтерів, мобільних пристроїв тощо.

Селектори, властивості та значення

Типи селекторів (клас, ідентифікатор, тег), синтаксис властивостей та значень, спадкування стилів та каскадність

Блочна модель

Вміст, поля, рамки та відступи, box-sizing для зміни розмірів блоків, гнучкі розміри (тільки width/height або max-width)

Адаптивний дизайн

Медіа-запити для умовного CSS, макети для різних розмірів екранів, резинова верстка за допомогою flex та grid

Ці теми розглядаються більш детально з практичними прикладами. Це допоможе краще зрозуміти основи CSS та можливості стилізації веб-сторінок.
