RESTful API та веб-сервіси

API (Application Programming Interface) - це інтерфейс для взаємодії програм між собою. REST - поширений стиль архітектури веб-API.

Розуміння REST архітектури

Огляд принципів REST: клієнт-сервер, безстановість, кешування, єдиний інтерфейс. Переваги у веб розробці.

Створення та використання API

Проектування, розгортання та тестування власних API. Використання Postman та інших інструментів. Документація.

Автентифікація та авторизація

Огляд OAuth, JWT, Basic Auth та інших підходів до автентифікації користувачів та захисту доступу до API.
