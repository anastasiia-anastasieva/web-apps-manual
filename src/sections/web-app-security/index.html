Безпека веб-додатків

Безпека - це важливий аспект при розробці будь-яких додатків, особливо веб-додатків, які доступні через Інтернет. Розглянемо основні теми цього розділу.

Основи кібербезпеки

Огляд актуальних загроз та методів захисту, управління доступом, фізичний та мережевий захист.

Захист від XSS та CSRF атак

Вразливості в веб-додатках, ін'єкції, крос-сайтовий скриптинг, підробка міжсайтових запитів та їх запобігання.

HTTPS і шифрування

Безпечна передача даних з використанням протоколу HTTPS, сертифікати, шифрування даних, огляд SSL/TLS.
